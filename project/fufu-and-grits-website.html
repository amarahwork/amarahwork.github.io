Fufu + Grits Website

Web Dev

Motion Graphics

Challenge

Goodjuju lacked an internet presence and needed a website to facilitate contact with clients and streamline application submissions for prospective team members. Shortly after the completion of this website, it needed to be updated to reflect the studio's rebrand to Fufu and Grits.

Solution

I created both the original and rebranded website in Webflow (as the studio wanted a low code solution that they could update as needed).

The website is responsive and made with accessibility guidelines in mind, including the addition of a flashing images warning.
