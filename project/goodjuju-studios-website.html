Challenge

Goodjuju lacked an internet presence and needed a website to facilitate contact with clients and streamline application submissions for prospective team members.

Solution

I adapted wireframes designed by Chindo Nkenke-Smith to create a responsive website built in Webflow, using existing assets, so that it could be handed to other designers and developers if necessary. Webflow also provided an easy way for the company to manage the online store. This website was also repurposed for a later rebrand.
